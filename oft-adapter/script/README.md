# DEPLOY GUIDE

## Mainnet

### Deploy OFTAdapter on ETH Mainnet

```bash
forge script script/DeployOFTAdapter.s.sol --rpc-url $RPC_ETH --account $DEPLOYER --broadcast --verify --verifier-url $RPC_ETH_SCAN --etherscan-api-key $KEY_ETH_SCAN --delay 30
```
